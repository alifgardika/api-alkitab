API Alkitab Indonesia adalah sebuah third-party backend Alkitab yang sumber bacaannya berasal dari sabda.org.

Sabda.org adalah sebuah Yayasan Kristen non-profit, non-komersial yang bergerak dalam bidang pelayanan media komputer dan internet di Indonesia.

Ini adalah proyek open-source siapa saja dapat membantu pengembangan dari API ini. Silahkan berkontribusi dalam pengembangan API ini dengan fork. dan buat issue baru dan pull-request untuk issue tersebut.

Source code dari API ini ditulis dengan bahasa pemrograman Go dan tersedia di Github https://github.com/erwindosianipar/api-alkitab.

Baca dokumentasi untuk cara penggunaan.
